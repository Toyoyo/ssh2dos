Detailed Description

Previously named ntddndis.h. But that caused problems when '-I.' is part of CFLAGS (caused this file to shaddow the system <ntddndis.h>).

++ BUILD Version: 0001 Copyright (c) 1990-1993 Microsoft Corporation Module Name: ntddndis.h Abstract: This is the include file that defines all constants and types for accessing the Network driver interface device. Author: Steve Wood (stevewo) 27-May-1990 Revision History: Adam Barr (adamba) 04-Nov-1992 added the correct values for NDIS 3.0. Jameel Hyder (jameelh) 01-Aug-95 added Pnp IoCTLs and structures

Kyle Brandon (kyleb) 09/24/96 added general co ndis oids.

Some OIDs also taken from WireShark.
