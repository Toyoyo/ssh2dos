Detailed Description

A djgpp-like DOS memory interface for Borland's DOS extender (32rtm). Only Borland's BCC32 v4.5+ supported.

by G. Vanem gvane.nosp@m.m@ya.nosp@m.hoo.n.nosp@m.o Nov 2002

Running PowerPak under plain DOS, the 32rtm extender tries to emulate a subset of the Win32 API. Running the same app. under Win32 uses the real Win32 API (from kernel32.dll etc.). Therefore we cannot use int386() etc., but must use native calls. This isn't finished yet (if at all possible).
