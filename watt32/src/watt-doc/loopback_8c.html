A simple loopback device. At the moment it handles ICMP Echo-request and UDP sink/discard packets. It pased up the rest.

Add protocol handlers to `loopback_handler' pointer as required.

The modified (src/dst IP swapped) ip packet is enqueued to the IP receive queue after loopback_device() returns.

The return value is: -1 if IP-packet should not be enqueued to IP input queue >0 length of (modified) IP-packet
